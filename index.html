SQL lekérdezés típusok

Insert Az INSERT parancs segítségével új adatokat adhatunk hozzá egy táblához. 
INSERT INTO tabla_neve (oszlop1, oszlop2) VALUES (ertek1, ertek2);
Select Az SELECT parancs segítségével lekérdezhetjük az adatokat egy táblából. 
SELECT * FROM tabla_neve;
Update Az UPDATE parancs segítségével módosíthatjuk az existáló adatokat egy táblában. 
UPDATE tabla_neve SET oszlop1 = ertek1 WHERE feltetel;
Delete Az DELETE parancs segítségével törölhetjük az adatokat egy táblából. 
DELETE FROM tabla_neve WHERE feltetel;

SQL korlátozások

Korlátozás Leírás Pelda (szintaxis)
ELSŐDLEGES KULCS Egyedi azonosító. Nem lehet üres ( NULL), és nem ismétlődhet. id INT PRIMARY KEY
Idegen kulcs Capsolat egy másik tablával. Biztosítja, hogy a hivatkozott adat létezzen. FOREIGN KEY (id) REFERENCES masik(id)
NEM NULL Kötelező kitölteni; az oszlop nem maradhat üresen. nev VARCHAR(50) NOT NULL
EGYEDÜLÁLLÓ Minden értéknek különbözőnek kell lennie az oszlopban. email VARCHAR(100) UNIQUE
ALAPÉRTELMEZETT Alapértelmezett értéket ad, ha a felhasználó nem ad meg semmit. statusz VARCHAR(10) DEFAULT 'aktív'
ELLENŐRZÉS Logikai feltételhez köti az adatbevitelt (pl. minimum érték). kor INT CHECK (kor >= 0)
