Adatbázis műveletek

Adatbevitel (INSERT)

Az INSERT parancs segítségével új adatokat adhatunk hozzá egy táblához.

INSERT INTO tabla_neve (oszlop1, oszlop2) VALUES (ertek1, ertek2);

Adatlekérdezés (SELECT)

A SELECT parancs segítségével lekérdezhetjük az adatokat egy vagy több táblából.

SELECT oszlop1, oszlop2 FROM tabla_neve WHERE feltetel;

Adatmódosítás (UPDATE)

Az UPDATE parancs segítségével módosíthatjuk a meglévő adatokat egy táblában.

UPDATE tabla_neve SET oszlop1 = ertek1 WHERE feltetel;

Adattörlés (DELETE)

A DELETE parancs segítségével törölhetjük a meglévő adatokat egy táblából.

DELETE FROM tabla_neve WHERE feltetel;
